CSS Selectors

How to call an element inside the styles

There are 6 types of selectors in css

  • Universal Selector (by using *)
  • Class name selector (by using .)
  • Id Selector (by using #)
  • Tag Name or element name selector (by using tag name)
  • Attribute selector (using any attribute of the html element)
  • Attribute with value selector (by using name of the attribute with that value)
