Combinators

1) Descendant selector (using space)
Selects all childs and grand childs in a parent element
2) Child selector (using > symbol)
Selects only direct childs of the parent element but not for grand childs
3) General Sibling Selctor ( using ~ symbol)
Selects all menctioned elements after the parent element
3) Adjacent Sibling Selctor ( using + symbol)
Selects single menctioned element after the parent element
