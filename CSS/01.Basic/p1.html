CSS (Cascading style sheets)

the css is used to apply the styles to html web pages or html documents

  1. Inline css (we can declare the inline css by using style attribite inside the any open html element and this inline css is only for single tag at a time)
  2. Internal css (we can declare the internal css using style tag inside the head tag and this internal css is for multiple tags but only for single html web page)
  3. External css (we can declare the External css by creating external css file by using .css extenction and linking that css file using link with 2 attributes that is rel ="stylesheet" and href ="address of the css files") inside the html head tag
